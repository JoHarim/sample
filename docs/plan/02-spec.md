# 기능명세서 — 스마트 알람 (가칭)

## 화면: S0 온보딩 (첫 실행 1회 — 알림 권한 + 도시 설정)
| No | 구분 | 기능 | 설명 | 우선순위 |
|---|---|---|---|---|
| 1 | 표시 | 환영 문구 | 앱 목적 한 줄 | MVP |
| 2 | 동작 | 알림 권한 요청 | 눌러 OS 권한창 → 허용/거부. 거부해도 진행("나중에 켤 수 있어요") | MVP |
| 3 | 입력 | 도시 선택 | 도시명 입력·검색 → 설정에 저장. 못 찾으면 재검색 안내 | MVP |
| 4 | 동작 | 시작 버튼 | 누르면 → S1 목록 | MVP |

## 화면: S1 알람 목록 / 홈 (내 알람을 보고 켜고 끈다)
| No | 구분 | 기능 | 설명 | 우선순위 |
|---|---|---|---|---|
| 1 | 표시 | 상단 도시·오늘 날씨 | 도시명 탭하면 변경 · 오늘 날씨 한 줄 | MVP |
| 2 | 표시 | 알림권한 경고 배너 | 권한 꺼져 있을 때만 "지금은 알람이 안 울려요" + 설정 이동 | MVP |
| 3 | 목록 | 알람 카드 | 시간·라벨·반복 요일·날씨조정 배지·켜기 토글. 시간순 정렬 | MVP |
| 4 | 동작 | 카드 탭 | 누르면 → S2 편집 | MVP |
| 5 | 동작 | + 버튼 | 누르면 → S2 추가 | MVP |
| 6 | 표시 | 빈 상태 | 알람 0개면 "첫 알람을 만들어 보세요" + 만들기 버튼 | MVP |

## 화면: S2 알람 추가·편집 (알람 하나를 만들거나 고친다)
| No | 구분 | 기능 | 설명 | 우선순위 |
|---|---|---|---|---|
| 1 | 입력 | 시간 선택 | 시:분 (필수) | MVP |
| 2 | 입력 | 반복 요일 | 월~일 토글 (하나 이상 필수) | MVP |
| 3 | 입력 | 라벨 | 이름(선택, 예: 출근 알람) | MVP |
| 4 | 입력 | 날씨 조정 켜기 | 켜면 "몇 분 일찍" 숫자 입력 노출(기본 15) | MVP |
| 5 | 입력 | 알람음·진동 | 알람음 고르기 · 진동 on/off | MVP |
| 6 | 동작 | 저장 | 시간 미선택·요일 0개면 막고 안내 → 성공 시 S1 | MVP |
| 7 | 동작 | 삭제 | (편집일 때만) 진짜 삭제 → S1 | MVP |

## 화면: S3 알람 울림 (알람이 울릴 때 확실히 깨운다)
| No | 구분 | 기능 | 설명 | 우선순위 |
|---|---|---|---|---|
| 1 | 표시 | 큰 현재 시각·라벨 | 화면 가득 | MVP |
| 2 | 표시 | 조정 안내 | "오늘 비 예보로 15분 일찍!" (조정된 알람일 때) | MVP |
| 3 | 표시 | 큰 소리·진동 | 울리는 동안 재생 | MVP |
| 4 | 동작 | 밀어서 끄기 | 슬라이더를 끝까지 밀어야 종료(스누즈 없음) | MVP |
