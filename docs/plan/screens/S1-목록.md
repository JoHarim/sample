# S1. 알람 목록 / 홈 — 내 알람 보고 켜고 끄기

## 들어오는 길
- 온보딩(S0) 완료 후 · 앱 재실행 시 첫 화면 · S2 저장/삭제 후 돌아옴

## 화면 구성 (위에서 아래로)
1. 상단 도시명(탭하면 변경) + 오늘 날씨 한 줄 → 2. (권한 꺼짐일 때만) 경고 배너 → 3. 알람 카드 목록(시간순 — 카드: 오전/오후 12시간제 시각·라벨·반복요일·소리/진동 여부·날씨조정 배지·켜기 토글) → 4. + 알람 추가 버튼

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 도시명 탭 | 도시 변경 입력 → settings 갱신 | 검색 실패 시 재시도 |
| 알람 켜기 토글 | 해당 알람 enabled 갱신 + 알림 재예약/해제 | 저장 실패 시 원상복구 + 안내 |
| 알람 카드 탭 | S2 편집으로 | — |
| + 버튼 | S2 추가로 | — |
| 경고 배너 탭 | OS 알림 설정으로 이동 | — |

## 예외 5종
| 상황 | 이 화면의 처리 |
|---|---|
| 비어 있을 때 (알람 0개) | "첫 알람을 만들어 보세요" + 만들기 버튼 |
| 로딩 중 | 상단 날씨 불러오는 동안 "날씨 확인 중" (알람 목록은 로컬이라 즉시) |
| 실패 (통신·저장 에러) | 날씨 못 불러오면 "날씨 정보를 못 불러왔어요" (알람은 원래 시각으로 동작) |
| 권한 없음 · 비로그인 | 알림 권한 꺼짐이면 상단 빨간 배너 "지금은 알람이 안 울려요" + 설정 이동 |
| 잘못된 입력 | 해당 없음 |

## 읽고 쓰는 데이터 (창고 연결)

- 읽기(외부): OpenWeatherMap(오늘 날씨 한 줄)
