# S0. 온보딩 — 알림 권한 + 도시 설정 (첫 실행 1회)

## 들어오는 길
- 앱 최초 실행 시 자동. 이미 온보딩을 마쳤으면 건너뛰고 바로 S1.

## 화면 구성 (위에서 아래로)
1. 환영 문구 → 2. 알림 권한 허용 버튼 → 3. 도시 입력·검색 → 4. 시작 버튼

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 알림 권한 허용 | OS 권한창 → 허용 시 다음 | 거부해도 진행("나중에 설정에서 켤 수 있어요") |
| 도시 검색·선택 | settings.city에 저장 | 못 찾으면 "도시를 못 찾았어요" + 재검색 |
| 시작 버튼 | S1 목록으로, 온보딩 완료 표시 저장 | 도시 미설정이면 "도시를 먼저 정해주세요" |

## 예외 5종
| 상황 | 이 화면의 처리 |
|---|---|
| 비어 있을 때 | 해당 없음(입력 화면) |
| 로딩 중 | 도시 검색 시 잠깐 "검색 중" |
| 실패 (통신·저장 에러) | 도시 검색 실패 시 "잠시 후 다시" + 재시도 |
| 권한 없음 · 비로그인 | 알림 권한 거부해도 진행 허용(로그인 없음) |
| 잘못된 입력 | 도시 미설정으로 시작 시 안내 |
