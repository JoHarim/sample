# S3. 알람 울림 — 울릴 때 확실히 깨우기

## 들어오는 길
- 예약된 알람 시각 도달 시 자동으로 뜸(폰 잠금 상태 포함). 날씨 조정이 켜져 있고 비/눈 예보면 N분 일찍 뜬다.

## 화면 구성 (위에서 아래로)
1. 큰 현재 시각 + 라벨 → 2. (조정된 알람이면) "오늘 비 예보로 15분 일찍!" → 3. 큰 소리·진동 재생 → 4. 밀어서 끄기 슬라이더

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 밀어서 끄기 | 끝까지 밀면 소리·진동 정지 + 종료(다시 안 울림) | 덜 밀면 원위치, 계속 울림 |

## 예외 5종
| 상황 | 이 화면의 처리 |
|---|---|
| 비어 있을 때 | 해당 없음 |
| 로딩 중 | 해당 없음 |
| 실패 (통신·저장 에러) | 핵심 안전선: 날씨 조회가 실패/오프라인이어도 알람은 원래 시각에 반드시 울린다(조정만 생략) |
| 권한 없음 · 비로그인 | 알림 권한이 없으면 이 화면 자체가 안 뜸 → S1 배너로 사전 경고 |
| 잘못된 입력 | 해당 없음(밀어서 끄기만) |

## 읽고 쓰는 데이터 (창고 연결)
- 읽기: alarms(울리는 알람의 라벨·소리·진동·조정 정보)
- 쓰기: 없음(스누즈 없음이라 상태 변경 없음)
