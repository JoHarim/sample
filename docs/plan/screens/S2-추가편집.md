# S2. 알람 추가·편집 — 알람 하나 만들기/고치기

## 들어오는 길
- S1의 + 버튼(추가) · S1의 알람 카드 탭(편집)

## 화면 구성 (위에서 아래로)
1. 시간 선택(시:분) → 2. 반복 요일 토글 → 3. 라벨(선택) → 4. 날씨 조정 토글 + "몇 분 일찍" → 5. 알람음·진동 → 6. 저장 (편집이면 7. 삭제)

## 행동과 흐름
| 행동 | 성공하면 | 실패하면 |
|---|---|---|
| 저장 | 폰 로컬 저장 + 알림 예약 → S1 | 시간 미선택·요일 0개면 막고 안내 / 저장 실패 시 재시도 |
| 날씨 조정 토글 | 켜면 "몇 분 일찍" 입력 노출(기본 15) | — |
| 삭제(편집만) | 진짜 삭제 + 예약 해제 → S1 | 삭제 실패 시 재시도 |

## 예외 5종
| 상황 | 이 화면의 처리 |
|---|---|
| 비어 있을 때 | 해당 없음(입력 화면) |
| 로딩 중 | 저장 시 잠깐 비활성 |
| 실패 (통신·저장 에러) | 저장/삭제 실패 시 "저장하지 못했어요" + 재시도 |
| 권한 없음 · 비로그인 | 로그인 없음. (알림 권한은 S1 배너로 별도 안내) |
| 잘못된 입력 | 시간 미선택 → "시간을 정해주세요" / 요일 0개 → "요일을 하나 이상 골라주세요" |
