## 한 문장 소개
아침에 잘 못 일어나는 사람을 위해 **모바일 알람 앱**을, 핵심 기능 **날씨에 맞춰 시각이 스스로 바뀌고(똑똑한 시간) 무심코 못 끄게 하는(확실히 깨우기)** 것으로 만든다.

## 데이터 (뭘 저장하나 — 폰 로컬 저장, MVP는 서버 없음)
- 표: **alarms** (필드: 라벨·시간·반복 요일·켜짐·날씨조정·몇 분 일찍·알람음·진동)
- 표: **settings** (1줄 — 도시·기본 알람음)
- 관계: 없음(settings는 전역 1줄). 상세는 docs/plan/05-db.md

## 조립 순서 (블럭 = 기능 하나 / 스텝 = 한 번에 시킬 일 하나)
> 블럭 규칙: 끝나면 눈에 보이는 결과가 하나 나온다 · 앞 블럭만 있으면 된다 · 이 블럭에 필요한 것만.
> 먼저 준비: pnpm-workspace.yaml에서 apps/mobile 주석 해제 → pnpm install (모바일 재료 하이드레이트).

### 블럭 1. 기본 알람 (완료 기준: 시간·요일을 정해 알람을 만들면 그 시각에 폰이 잠겨 있어도 소리·진동 알림이 온다)
- [x] 1-1. 알람 목록(홈) 화면 골격 — 확인: 앱에 빈 목록 화면이 뜬다 (명세: docs/plan/screens/S1-목록.md)
- [x] 1-2. 알람 추가 화면 폼(시간·요일·라벨) — 확인: 시간·요일 고르고 저장 버튼이 눌린다 (명세: docs/plan/screens/S2-추가편집.md)
- [x] 1-3. 폰 로컬 저장 + 목록 표시 — 확인: 저장하면 목록에 뜨고, 앱 껐다 켜도 남아 있다
- [ ] 1-4. 알림 예약·울리기 + 알림 권한 요청 — 확인: 1~2분 뒤로 맞추면 폰에서 소리·진동 알림이 온다 (명세: docs/plan/screens/S0-온보딩.md)

### 블럭 2. 확실히 깨우기 (완료 기준: 알람이 울리면 화면을 끝까지 밀어야 꺼지고 큰 소리·진동이 난다)
- [ ] 2-1. 알람 울림 화면 + 밀어서 끄기 슬라이더 — 확인: 알람이 뜨면 끝까지 밀어야만 꺼진다 (명세: docs/plan/screens/S3-울림.md)
- [ ] 2-2. 큰 소리·진동 재생 — 확인: 울릴 때 소리와 진동이 난다
- [ ] 2-3. 스누즈 없음(밀면 완전 종료) — 확인: 끈 뒤 다시 울리지 않는다

### 블럭 3. 날씨 따라 조정 (완료 기준: 비/눈 예보면 설정 시각보다 자동으로 N분 일찍 울리고, 조정된 시각을 목록에 보여준다)
- [ ] 3-1. 도시 설정(온보딩 + 목록 상단) — 확인: 도시를 정하면 저장된다 (명세: docs/plan/screens/S0-온보딩.md)
- [ ] 3-2. OpenWeatherMap 연결(도시 예보 조회) — 확인: 오늘 비/눈 여부를 읽어온다
- [ ] 3-3. 알람에 날씨조정 켜기 + 몇 분 일찍 — 확인: 알람에 조정 옵션이 저장된다
- [ ] 3-4. 조정 적용 + 목록 표시 — 확인: 비 예보면 알람이 N분 일찍 예약되고 목록에 "비 예보로 15분 일찍"이 뜬다
- [ ] 3-5. 안전선: 날씨 실패해도 원래 시각에 울림 — 확인: 인터넷을 끄고도 알람은 원래 시각에 울린다
